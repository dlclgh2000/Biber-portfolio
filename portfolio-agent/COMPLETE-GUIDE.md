# 🎨 AI 포트폴리오 에이전트 - 완전 가이드

## 🌟 개요

이 에이전트는 **피그마 링크**와 **프로젝트 설명**만 제공하면, Claude Code가 자동으로 **시니어 프로덕트 디자이너 수준의 전문적인 포트폴리오**를 작성해주는 도구입니다.

### 기존 방식 vs AI 에이전트

| 기존 방식 | AI 에이전트 |
|---------|-----------|
| 수작업으로 HTML 작성 | 자동 HTML 생성 |
| 포트폴리오 텍스트 고민 | AI가 전문적으로 작성 |
| 일관성 없는 구조 | 표준화된 케이스 스터디 |
| 시간 소요: 2-3시간 | 시간 소요: 5분 |

## 🚀 사용법 (3단계)

### 1단계: 정보 입력 (2분)

```bash
node create-portfolio.mjs
```

대화형으로 다음을 입력:
- 프로젝트 ID
- 제목/부제목
- 피그마 링크 (있으면)
- 프로젝트 설명
- 이미지 경로

### 2단계: Claude에게 전달 (1분)

스크립트가 생성하는 메시지를 복사해서 이 대화창에 붙여넣기:

```
portfolio-request-{id}.json 파일을 읽고,
시니어 프로덕트 디자이너 관점에서 전문적인 포트폴리오를 작성해주세요.
...
```

### 3단계: 완료 (2분)

Claude가 자동으로:
1. ✅ 피그마 디자인 분석
2. ✅ 전문 포트폴리오 작성
3. ✅ HTML 파일 생성
4. ✅ 데이터베이스 업데이트
5. ✅ 메인 페이지 업데이트

## 🎯 생성되는 포트폴리오 구성

Claude가 다음 8개 섹션으로 구성된 전문 케이스 스터디를 작성합니다:

```
┌─────────────────────────────────────────┐
│  1. 프로젝트 개요                       │
│     → 핵심 목표와 배경을 명확하게       │
├─────────────────────────────────────────┤
│  2. 문제 정의                           │
│     → 사용자/비즈니스 과제 분석         │
├─────────────────────────────────────────┤
│  3. 디자인 목표                         │
│     → 3-5개 핵심 목표                   │
├─────────────────────────────────────────┤
│  4. 주요 기능                           │
│     → 각 기능의 목적과 구현 방법        │
├─────────────────────────────────────────┤
│  5. 디자인 프로세스                     │
│     → 리서치부터 반복까지 전 과정       │
├─────────────────────────────────────────┤
│  6. 디자인 결정                         │
│     → 주요 결정과 근거                  │
├─────────────────────────────────────────┤
│  7. 성과 및 임팩트                      │
│     → 측정 가능한 결과와 피드백         │
├─────────────────────────────────────────┤
│  8. 배운 점                             │
│     → 프로젝트 인사이트                 │
└─────────────────────────────────────────┘
```

## 💎 핵심 기능

### 🎨 피그마 디자인 분석

피그마 링크를 제공하면:
- UI 컴포넌트 자동 분석
- 디자인 시스템 파악
- 색상/타이포그래피 추출
- 레이아웃 패턴 인식

### 🤖 AI 기반 전문 작성

Claude가 시니어 디자이너 관점에서:
- UX 전략 설명
- 사용자 중심 사고 반영
- 데이터 기반 의사결정 강조
- 비즈니스 임팩트 강조

### 📄 자동 HTML 생성

반응형 디자인의 아름다운 HTML:
- 모바일/데스크톱 지원
- AOS 애니메이션
- 깔끔한 타이포그래피
- 일관된 스타일

### 🔄 완전 자동화

한 번의 실행으로:
- `detail-{id}.html` 생성
- `projects-data.json` 업데이트
- `index.html` 갱신
- 백업 자동 생성

## 🔧 고급 기능

```bash

code detail-project-id.html

# 2. 또는 JSON 수정 후 재생성
code portfolio-request-project-id.json
# Claude에게 다시 요청
```


```bash
node portfolio-manager.js delete project-id
node portfolio-manager.js generate
```

## 🆘 문제 해결

### Q1: "파일을 찾을 수 없습니다"

```bash
# 현재 디렉토리 확인
pwd
# Biber-portfolio 폴더여야 함

# 파일 존재 확인
ls *.mjs
```

### Q2: "Claude가 응답하지 않아요"

- 메시지를 정확히 복사했는지 확인
- JSON 파일이 생성되었는지 확인: `ls portfolio-request-*.json`

### Q3: "생성된 포트폴리오가 마음에 안 들어요"

```bash
# 1. JSON 파일에서 설명을 더 구체적으로 수정
code portfolio-request-project-id.json

# 2. Claude에게 재요청
"더 전문적이고 데이터 중심적으로 작성해주세요"
```

### Q4: "HTML이 깨졌어요"

```bash
# 백업에서 복원
ls index.backup.*.html
cp index.backup.1234567890.html index.html
```
